Function errorHandler

  • errorHandler(
        err: ICustomError,
        req: Request,
        res: Response,

    Express middleware for handling exceptions that occur during the processing of requests. This error handler logs the error to the console and sends a JSON response with the error message and appropriate status code.

    It checks for a statusCode property on the error object to determine the HTTP response status. If none is found, it defaults to 500, indicating an Internal Server Error. If there's no error, it simply passes control to the next middleware.

    Parameters

    • err: ICustomError

      The error object that may contain custom properties such as statusCode.

    • req: Request

      Express Request object. Not used in this function, but necessary for middleware signature.

    • res: Response

      Express Response object used to send back the error response.

    • next: NextFunction

      Callback to pass control to the next middleware function in the stack.

    Returns void
